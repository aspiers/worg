CSS - body

Choose a big font! 

body
{
  width:100%;
  padding:0px 0px 0px 0px;
  margin:0px 0px 0px 0px;
  background-image:url(img/bg.gif);
  background-attachment:fixed;
  background-repeat:repeat-x;
  /* ... etc. */
}

CSS - headlines

Set all headlines to the same size: 

h1,h2,h3,h4,h5,h6
{
  font-size: 18pt;
  font-weight:bold;
  color:#104d8c;
  /* ... etc. */
}

CSS - org-info-js

Position the org-info.js window. 

#org-info-js-window
{
  position:absolute;
  top:140px;
  left:20%;
}
